# Globe texture

The hero globe uses `public/Images/Illustrations 2/illustration-global-inspiration-map.png` (rasterized from the matching SVG for WebGL), configured in `main.ts`. The circle stroke around the globe is `illustration-global-outline-circle.svg`, overlaid in the hero.

For a custom flat map PNG (equirectangular 2:1, e.g. 2048×1024), save it under `public/globe/` and point `main.ts` at that path instead.
